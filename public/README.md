## Tech stack
- HTML, CSS, JavaScript
- Optional: a frontend framework (React / Vue / Svelte) or static site generator
- Optional build tools: Node.js, npm/yarn, Vite, webpack
- Optional: CI/CD provider and hosting (GitHub Pages, Netlify, Vercel)
